# Tasty TedoZ

**Tasty TedoZ** is a simple and welcoming website project built to showcase a restaurant-style brand online. It includes pages for home, menu/services, about, blog, and contact, with supporting styles and assets.

## What this project contains

- `index.html` — the home page where visitors first arrive.
- `About.html` — information about the restaurant, its story, and values.
- `Services.html` — details on what the restaurant offers, such as food, delivery, or special services.
- `Blog.html` — a page for news, updates, recipes, or dining stories.
- `Contact.html` — a place where users can find contact details and reach out.
- `assets/css/style.css` — the main stylesheet used by the site.

## Purpose

This repo is intended to present a clean, easy-to-read restaurant website layout that anyone can understand and explore. It is useful as:

- a learning example for HTML and CSS
- a simple website template for a restaurant or food business
- a practice project for building multi-page sites

## How to use it

1. Open the project folder in your code editor.
2. Open `index.html` in a browser to view the website.
3. Edit the HTML files or `assets/css/style.css` to customize the content and design.

## Who can use it

Anyone can use this project:

- students learning web development
- designers wanting a lightweight restaurant site
- developers creating an easy demo page

## Notes

- All pages are static HTML, so no server or database is required.
- The styling is managed in `assets/css/style.css`.
- You can copy, edit, or expand the pages to fit your own restaurant or food brand.

## Summary

This repo is a simple restaurant website called **Tasty TedoZ**. It is built with plain HTML and CSS and is easy for anyone to understand, edit, and use.
